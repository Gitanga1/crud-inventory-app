# 📦 Inventory Management System

A full-stack CRUD web application for local SMEs to manage products, track stock levels, and export data.

## 🚀 Live Demo
**https://crud-inventory-app.onrender.com**

## 📋 Features
- Add new products with name, price, and quantity
- View all products in a table
- Edit product details inline
- Delete products (password protected)
- Search products by name
- Low stock alerts (quantity < 5)
- Export data to CSV
- Cloud database with Supabase

## 🛠️ Technology Stack
- Python 3.14, Flask 3.0.0
- PostgreSQL (Supabase)
- SQLAlchemy ORM
- HTML5, CSS3, Jinja2
- Render.com (deployment)

## 💻 Local Development

\`\`\`bash
git clone https://github.com/Gitanga1/crud-inventory-app.git
cd crud-inventory-app
pip install -r requirements.txt
python app.py
\`\`\`
